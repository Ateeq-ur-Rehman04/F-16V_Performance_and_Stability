# 2-Wheeled Self-Balancing Obstacle-Avoiding Robot

Softwares and Tools used: Arduino, MATLAB/Simulink, LabView, VS Code

A custom-built 2-wheeled self-balancing obstacle avoiding robot that not only balances itself upright in, both statically and upon any disturbance, but moves forward avoiding any impending obstacles, all while balancing itself.
The mechanics are modelled using Euler-Langragian equations and PID is implemented for control.
Real-time telemetry is also inorporated using MATLAB, Simulink, LabView and Python for live motion tracking and data capturing.
